Get Started
Two commands. That's it.
Install the CLI, pick your AI assistant, and the skill is ready. Works with Claude Code, Cursor, Windsurf, and 15 more.
# Install the CLI npm install -g ui-ux-pro-max-cli
# Pick your AI assistant uipro init --ai claude # Claude Code uipro init --ai cursor # Cursor uipro init --ai windsurf # Windsurf uipro init --ai copilot # GitHub Copilot uipro init --ai all # All assistants
